# Siegers ホームページ 現状要件(current-state)

草野球チーム「Siegers」のホームページが現在満たしている要件の一覧。

- 更新日: 2026-07-12
- このドキュメントは常に「実装済みの現状」を表す。新しい要件が実装に取り込まれたら必ずここに反映する。
- 技術構成・ページ構成の詳細は [docs/design/current-state.md](../design/current-state.md) を参照。

## 対象ユーザー

- **一般訪問者**: チームメンバー・対戦相手・見学希望者など。閲覧のみ
- **管理者**: チームの成績入力担当。ログインして試合・選手・設定を管理する

## 機能要件(実装済み)

### 公開機能(誰でも閲覧可)

- **FR-1 トップページ**: チームの通算勝敗数と最近の試合結果を表示する
- **FR-2 試合一覧**: 日付・対戦相手・スコア・勝敗・球場を一覧表示する。フィルターで絞り込みできる
- **FR-3 試合詳細**: イニングスコア・出場選手の打撃成績・投手成績を表示する
- **FR-4 選手通算成績**: 全選手の打撃・投手の通算成績と率系指標(打率・防御率・OPS など)を表示する。規定打席・規定投球回による絞り込みに対応する
- **FR-5 選手個人ページ**: 選手ごとの成績を表示する
- **FR-6 チーム紹介**: チームの紹介情報を表示する

### 管理機能(認証必須)

- **FR-7 ログイン**: メールアドレス + パスワードで管理者を認証する
- **FR-8 試合結果の入力**: フォーム入力または JSON 一括入力で試合と成績を登録する
- **FR-9 試合結果の編集・削除**: 登録済みの試合を編集・削除する
- **FR-10 選手管理**: 選手の登録・削除・投手フラグの切替を行う
- **FR-11 マスタ管理**: 対戦相手・大会名のマスタを管理し、試合入力時の候補に使う
- **FR-12 設定**: 規定打席・規定投球回の倍率を変更する

## 非機能要件(実装済み)

- **NFR-1 認可**: 未ログインで `/admin` 配下にアクセスすると `/admin/login` にリダイレクトする。DB は RLS で「閲覧は全員可・書き込みは認証済みのみ」を保証する
- **NFR-2 スマホ対応**: 全ページをスマートフォンで閲覧・操作できる(成績テーブルは横スクロール + 選手名列固定)
- **NFR-3 データ取得の完全性**: 一覧取得は Supabase の 1000 行制限を回避して全件取得する

## 制約・前提条件

- ホスティングは Vercel、DB・認証は Supabase を利用する
- DB 変更はマイグレーションツールを使わず、SQL を手動適用する運用
- 管理者は少人数(チーム内の担当者)を想定し、権限のロール分けは行わない
